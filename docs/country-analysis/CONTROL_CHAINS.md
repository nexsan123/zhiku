## 链1：能源链

```
谁产 ──→ 谁运 ──→ 谁买 ──→ 咽喉在哪 ──→ 谁能切断

石油：
  产：美国(#1)/沙特(#2)/俄罗斯(#3)/加拿大/巴西/UAE/伊拉克/伊朗
  运：油轮（希腊/中国船队）+ 管道（中俄ESPO/中缅/中哈）
  咽喉：霍尔木兹海峡（全球20%石油）/ 马六甲（中国80%海运油）/ 苏伊士
  买：中国(#1进口)/印度(#2)/欧洲/日本/韩国
  定价：OPEC+（沙特主导）+ 期货市场（WTI/布伦特）

天然气：
  产：美国(#1)/俄罗斯(#2)/卡塔尔(#3 LNG)
  运：管道（俄→欧/俄→中/中亚→中）+ LNG船
  买：欧洲（从俄气→美LNG）/中国(#1 LNG进口)/日本(#2)
  定价：TTF（欧洲）/ HH（美国）/ JKM（亚洲）

电力/新能源：
  设备产：中国（光伏80%/电池60%/风机主导）
  消费：全球
  控制权：中国通过设备制造控制全球能源转型速度
```

### 能源链上的权力关系

| 节点 | 控制者 | 被控制者 | 杠杆 |
|------|--------|---------|------|
| 油价定价 | 沙特/OPEC+ | 所有进口国 | 增减产→油价±30% |
| 霍尔木兹海峡 | 伊朗（地理）/美国（海军） | 全球20%石油通过 | 封锁→油价$150+ |
| 马六甲海峡 | 美国海军（但中国A2/AD覆盖扩大） | 中国/日本/韩国 | 军事封锁可行性在降低 |
| LNG供应 | 美国（对欧57%） | 欧洲 | 美国LNG=新能源杠杆 |
| 能源转型设备 | **中国**（光伏80%/电池60%） | **全球所有想转型的国家** | 不买中国设备→转型减速 |
| 石油结算货币 | 美国（美元）↔中国（人民币进攻） | — | **沙特选哪边=美元体系的开关** |

### 量化触发点

| 事件 | 链上传导 | 市场信号 |
|------|---------|---------|
| OPEC+意外减产 | 供应↓→油价↑→通胀↑→央行鹰派 | 布伦特↑/航空股↓/能源股↑ |
| 霍尔木兹冲突 | 20%石油中断→油价暴涨→全球衰退 | 原油↑50%+/全球股指↓/黄金↑/VIX↑↑ |
| 中国限制光伏出口 | 全球能源转型减速→碳排放↑→碳价波动 | 欧美光伏股↓/中国光伏股↑/碳期货波动 |
| 沙特增加人民币结算 | 美元需求↓→DXY↓ | DXY↓/黄金↑/人民币↑/美债收益率↑ |

## 链2：技术链

```
谁设计 ──→ 谁造设备 ──→ 谁制造 ──→ 谁封锁 ──→ 替代方案

芯片：
  设计：美国（高通/英伟达/AMD/苹果）+ 中国追赶（华为海思）
  EDA软件：美国垄断（Synopsys/Cadence）
  光刻机：荷兰ASML（EUV全球唯一）
  设备：日本32% + 美国 + 荷兰
  材料：日本（硅晶圆90%/光刻胶50%）
  制造：台湾TSMC（先进制程90%）+ 韩国Samsung + 中国SMIC（7nm推量）
  封装：中国AMIES（中国市场90%，全球35%）

AI：
  模型：美国（OpenAI/Google/Anthropic）+ 中国（DeepSeek/阿里/字节）
  算力：美国（74.5% GPU集群）+ 中国（推量+能源优势）
  数据：美国（英语互联网）+ 中国（14亿用户数据）
  能源基础：中国优势（便宜电力+光伏/电池/核电自控）
```

### 技术链上的权力关系

| 节点 | 控制者 | 被控制者 | 不可替代性 |
|------|--------|---------|-----------|
| EUV光刻机 | ASML（荷兰/被美国指挥） | 所有芯片制造商 | **全球唯一**（但中国在追） |
| 芯片设计IP | 美国 | 全球 | 极高（EDA+架构） |
| 硅晶圆 | 日本（信越+SUMCO 90%） | 所有晶圆厂 | **全球垄断** |
| 先进制程制造 | 台湾TSMC（90%） | 美国/全球 | 极高（CHIPS Act是备份非替代） |
| AI算力 | 美国短期（GPU数量） | — | 高但中国能源优势在追 |
| AI算法效率 | **趋向平衡**（DeepSeek） | — | 不再是美国独占 |

### 量化触发点

| 事件 | 链上传导 | 市场信号 |
|------|---------|---------|
| 台海冲突 | TSMC停产→全球芯片断供→手机/汽车/AI全停 | 半导体股崩↓50%+/避险资产暴涨 |
| ASML被限制对中国出口更多型号 | 中国芯片受阻→但SMIC推量加速→长期中国自主化 | ASML↓（失去中国收入）/SMIC概念↑ |
| DeepSeek发布新突破 | 算法效率提升→对Nvidia GPU需求减少 | Nvidia↓（2025.1已蒸发$5930亿） |
| 中国EUV原型机取得进展 | ASML垄断松动信号 | ASML长期估值重估 |

## 链3：金融链

```
谁发行货币 ──→ 谁控制结算 ──→ 谁制裁 ──→ 谁在建替代 ──→ 替代进度

当前体系（美元主导）：
  美元：全球储备57% / 外汇交易88% / 石油结算主导
  SWIFT：美国有否决权（名义在布鲁塞尔）
  制裁工具：SDN名单/二级制裁/冻结资产/SWIFT踢除

平行体系（中国建设中）：
  CIPS：1683家机构/180+国家/年交易$24.5万亿/年增42.6%
  e-CNY：$2.3万亿累计交易/+800%增长
  人民币储备：3%（缓慢增长）
  人民币贸易融资：全球#2
  mBridge：$554.9亿（+2500倍自2022）
```

### 金融链上的权力关系

| 节点 | 控制者 | 武器化程度 | 替代进度 |
|------|--------|-----------|---------|
| 全球储备货币 | 美元57% | 极高（可冻结任何国家资产） | 缓慢侵蚀（10-15年维度） |
| 跨境支付 | SWIFT（美国控制） | 极高（踢除=金融死刑） | CIPS年增42.6%但仍小 |
| 石油结算 | 美元（但沙特开始接受人民币） | 极高（石油美元=美元需求根基） | 裂缝已出现 |
| 制裁执行 | 美国（二级制裁域外管辖） | 越用越钝（中俄伊已建绕过体系） | 效力下降中 |
| 数字货币 | 中国e-CNY领先 | 潜在（可绕过SWIFT） | 早期但增长快 |

### 量化触发点

| 事件 | 链上传导 | 市场信号 |
|------|---------|---------|
| 美元储备份额跌破50% | 美元信心拐点→利率↑→债务螺旋加速 | DXY↓↓/黄金↑↑/美债抛售/新兴市场货币波动 |
| 美国对中国银行实施SWIFT制裁 | $3万亿贸易流中断→全球供应链断裂 | 全球股指暴跌/VIX>50/避险全面开启 |
| CIPS交易量超过SWIFT的10% | 平行体系临界质量 | 人民币资产重估↑/美元长期预期↓ |
| 沙特>30%石油用人民币结算 | 石油美元体系实质性松动 | DXY↓/黄金↑/人民币↑ |

## 链5：矿产链

```
谁挖 ──→ 谁加工 ──→ 谁消费 ──→ 卡脖子环节 ──→ 替代时间

稀土：中国挖60%+加工90% → 全球消费 → 卡在加工（西方替代10年+）
锂：澳/智利/阿根廷挖 → 中国加工65% → 全球电池 → 卡在加工
钴：刚果(金)挖70% → 中国加工73%（中资控制15/19矿）→ 全球电池
铁矿石：澳大利亚54%出口 → 中国消费73% → 钢铁 → Simandou改变格局
铜：智利/秘鲁挖 → 中国精炼45% → 全球 → 电气化时代需求↑↑
镓/锗：中国精炼99%/60% → 全球半导体 → 已被中国武器化
铂族：南非70%+ → 全球催化剂/氢能 → 南非独家
```

### 矿产链上的权力关系

| 节点 | 控制者 | 被控制者 | 武器化状态 |
|------|--------|---------|-----------|
| 稀土加工 | **中国90%** | 美国/欧洲/日本 | ✅ 已武器化（2025出口管制→欧价涨6倍） |
| 镓/锗精炼 | **中国99%/60%** | 全球半导体/光纤 | ✅ 已武器化（2024对美禁令） |
| 锂加工 | **中国65%** | 全球电池制造 | ✅ 2025.10出口管制 |
| 铁矿石供应 | 澳大利亚54%→被Simandou削弱 | 中国（73%进口方） | 中国是买方垄断→澳依赖中国 |
| 钴原矿 | 刚果(金)→**中资控制15/19矿** | 全球电池 | 中国通过所有权控制 |
| 铂族 | 南非70%+ | 全球催化剂/氢能 | 未武器化但潜在杠杆 |

## 5条链的交叉节点

**最危险的交叉点 = 多条链同时被影响的事件**

| 事件 | 影响的链 | 系统性风险级别 |
|------|---------|-------------|
| **台海冲突** | 技术链（TSMC断供）+ 能源链（马六甲）+ 金融链（制裁/反制裁）+ 矿产链（稀土反制） | **P0 最高级** |
| **美元信心崩塌** | 金融链（储备/结算/制裁全面失效）+ 能源链（石油定价重组）+ 粮食链（贸易结算混乱） | **P0** |
| **霍尔木兹封锁** | 能源链（20%石油）+ 金融链（油价→通胀→加息→债务）+ 粮食链（化肥→粮价） | **P1** |
| **中国全面稀土禁运** | 矿产链 + 技术链（芯片材料）+ 能源链（风机磁铁）| **P1** |
| **全球粮食危机** | 粮食链 + 金融链（通胀）+ 政治稳定（中东/北非革命）| **P1** |

## 对量化系统的直接价值

```
5条控制链 → AI推理引擎可以做"链式推导"：

输入：某条链上某个节点发生事件
推导：沿链传导 → 经过哪些国家 → 影响哪些市场
输出：具体的交易信号建议

示例：
  输入：OPEC+宣布减产200万桶/日
  链：能源链
  传导：供应↓ → 布伦特↑$15-20 → 全球通胀预期↑ → 债券收益率↑
        → 航空/运输成本↑ → 航空股↓
        → 能源出口国(沙特/俄/加/巴)收入↑ → 货币↑
        → 能源进口国(日/韩/印/欧)成本↑ → 货币↓
        → 中国：石油进口成本↑但化工原料型→影响有限
  信号：多布伦特/多能源股/空航空股/多沙特/空日元
```
