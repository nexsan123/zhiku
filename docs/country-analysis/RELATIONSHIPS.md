## 关系分类

| 类型 | 定义 | 典型标志 |
|------|------|---------|
| **结构性对抗** | 核心利益冲突，无法调和 | 军备竞赛/技术封锁/代理人战争 |
| **竞合关系** | 竞争为主但保持经贸联系 | 一边制裁一边$700B贸易 |
| **非对称依附** | 一方依赖另一方安全/经济 | 安全外包/资源绑定 |
| **利益联姻** | 无感情纽带，纯利益交换 | 你卖油我建基建 |
| **摇摆博弈** | 被多方争夺，利用位置获利 | 两边拿好处不选边 |
| **隐性合作** | 公开保持距离，暗地深度捆绑 | 影子贸易/非公开协议 |

| 金融 | 人民币国际化的练兵场 | 绕过西方制裁的通道 |

| 隐性博弈 | 说明 |
|---------|------|
| 力量2号管道 | 中国故意拖延→压低价格→俄罗斯越来越急 |

| 翻转信号 | 市场影响 |

| 俄罗斯政权更迭 | 新政府可能重新向西→中俄关系重置 |

**量化信号**：USDCNY / 乌拉尔原油折价 / 中俄贸易额 / 人民币在俄跨境支付占比

### 3. 🇺🇸美国 ↔ 🇸🇦沙特 | 解体中的联姻

```
表面：传统盟友（石油美元体系+安全保护）
真实：沙特正在从"附庸"变成"独立摇摆国"，美国的控制力在松动
```

| 维度 | 历史交换 | 2025变化 |
|------|---------|---------|
| 石油 | 沙特用美元卖油→维持美元体系 | 沙特开始接受人民币结算 |
| 安全 | 美国提供军事保护→沙特买美国武器 | 沙特和中国军事合作↑+和伊朗和解（中国调解） |
| 政治 | 美国忽略人权→沙特忠诚 | 美国批评人权→沙特不爽→找替代靠山 |
| 联盟 | 沙特是中东反伊朗的支柱 | 沙特加入BRICS+和伊朗建交→不再做美国的棋子 |

**这条关系对美元体系的意义**：
```
石油美元体系 = 全球石油用美元结算 → 所有国家必须持有美元 → 美元需求 → 美国可以印钱不通胀

沙特每增加1%人民币结算 = 美元需求下降 = 美元体系松动一分
这不是经济问题，是美国霸权的根基问题
```

| 翻转信号 | 市场影响 |
|---------|---------|
| 沙特宣布>30%石油接受人民币 | **美元指数暴跌/黄金暴涨/美债被抛/全球重定价** |
| 沙特加入中国安全协议 | 美国中东战略崩溃/军工股波动/油价不确定性↑↑ |
| 美国武力威胁沙特（极低概率） | 全球石油断供恐慌/油价$150+ |

**量化信号**：布伦特油价 / OPEC+产量决策 / 沙特对华石油出口量 / 人民币在沙特贸易中占比

### 4. 🇺🇸美国 ↔ 🇪🇺欧洲 | 主从关系 + 裂缝扩大

```
表面：最强联盟（NATO+价值观+历史）

     美国要求欧洲"选边"（对华/对俄），欧洲想两头通吃
```

| 维度 | 美国得到 | 欧洲得到 |
|------|---------|---------|

| 能源 | LNG出口市场（从俄气缺口填入） | 能源供应（但比俄管道气贵） |
| 贸易 | 欧洲市场 | 美国市场（但被关税威胁） |
| 对华 | 欧洲配合芯片管制（ASML禁售） | 被中国报复（稀土涨6倍/车企停产） |

| 隐性裂缝 | 说明 |
|---------|------|
| 能源成本 | 美国LNG比俄管道气贵→欧洲工业竞争力下降→德国化工外迁 |
| 特朗普因素 | 每4年可能上台一次→要求5% GDP军费→欧洲不堪重负 |
| ASML | 美国命令荷兰禁止对华出口→荷兰/欧盟失去中国市场→利益受损但不敢拒绝 |
| 对华政策 | 法国想自主/德国想贸易/东欧跟美国→内部分裂 |

**量化信号**：EURUSD / TTF天然气价格 / ASML股价+订单 / NATO军费达标率
