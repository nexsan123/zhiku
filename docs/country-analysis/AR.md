# 阿根廷 (AR) 战略画像

> 精简版 · 服务于智库量化系统

## 角色：周期性危机国 + 锂三角成员 + Milei实验观察窗

**一句话**：反复债务违约→IMF救助→民粹→再违约的循环国家。2024年Milei的激进自由化改革是近年最极端的经济实验。拥有全球前列的锂储量。

## 独特价值
- **锂**：全球储量#3（锂三角：阿根廷/智利/玻利维亚）→ 电池时代的战略资源
- **Vaca Muerta**：全球第二大页岩气+第四大页岩油储量
- **农业**：大豆/牛肉/小麦出口
- **Milei实验**：全球观察"极端自由化改革"能不能救一个烂国家

## 关键风险
- Milei改革失败→社会动荡→又一次违约
- 锂矿开发受政治/环保/社区阻力
- 比索黑市汇率与官方汇率双轨→真实经济状况不透明
