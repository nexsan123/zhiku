# 事件→因果→市场传导模板

> 核心用途：当事件发生时，AI推理引擎沿控制链+关系矩阵推导后果→输出交易信号
> 每个事件模板包含：触发条件→发生概率→即时反应→二阶后果→三阶后果→市场传导→监控信号
> 概率基于当前局势（2025-2026），会随时间变化，需定期校准

---

## 概率标注说明

| 标签 | 概率范围 | 含义 |
|------|---------|------|
| P-极高 | >70% | 几乎确定会发生，问的是"何时"不是"会不会" |
| P-高 | 40-70% | 可能发生，已有明确趋势 |
| P-中 | 15-40% | 有可能但需要特定触发条件 |
| P-低 | 5-15% | 不太可能但不能排除 |
| P-极低 | <5% | 黑天鹅级别 |

### E-001：台海军事冲突

| 维度 | 内容 |
|------|------|
| **发生概率** | **P-低~中（10-20%，未来5年窗口）** |
| 概率依据 | 中国军事能力窗口2027-2035最优；但经济代价极高（马六甲风险+制裁+供应链断裂）；理性计算下动武的净收益为负 |
| 触发条件 | 台湾宣布法理独立 / 美国部署军队到台湾 / 中国国内危机需要转移注意力 |
| 影响链 | 技术链+能源链+金融链+矿产链（4链同断） |

**传导推演**：

```
T+0（即时）：
  → TSMC停产（全球90%先进芯片断供）   [确定性: 95%]
  → 台湾海峡航运中断（全球50%集装箱船经过）  [确定性: 90%]
  → 全球股市暴跌（-15~30%数日内）   [确定性: 90%]
  → VIX > 60   [确定性: 85%]
  → 黄金突破$3500+   [确定性: 80%]
  → 油价飙升至$120+（海运中断恐慌）   [确定性: 75%]

T+1周（一阶后果）：
  → 美国/日本/澳宣布制裁中国   [概率: 85%]
  → 中国反制：稀土全面禁运   [概率: 70%]
  → 中国反制：冻结/抛售美债   [概率: 40%]
  → 保险公司撤销亚太航运保险   [概率: 80%]
  → 全球芯片库存开始消耗（存量约3-6个月）   [确定性: 90%]

T+1月（二阶后果）：
  → 全球汽车/手机/AI服务器产量骤降   [概率: 85%]
  → 欧洲车企因稀土断供+芯片断供双重打击→部分停产   [概率: 75%]
  → 日本经济衰退（失去中国贸易+台湾芯片双重打击）   [概率: 80%]
  → 韩国被迫选边（Samsung/SK Hynix产线调整）   [概率: 75%]
  → 印度宣布"不选边"但暗中增加对俄购油   [概率: 70%]
  → 沙特加速人民币结算（趁美元动荡分散风险）   [概率: 55%]

T+6月（三阶后果）：
  → 场景A（停火/冻结，概率45%）：
    军事对峙但未全面开战→封锁式消耗→双方寻找台阶
    → 市场部分恢复但"台湾风险溢价"永久存在
    → 全球加速芯片分散化（TSMC美日厂扩产+中国SMIC推量加速）

  → 场景B（全面冲突，概率25%）：
    美中直接军事交战→全球经济衰退
    → GDP损失：全球-5~10%（Bloomberg估算）
    → 中国GDP -15~25%（海运断+制裁+外资撤离）
    → 美国GDP -5~10%（芯片断+供应链断+通胀飙升）
    → 可能持续1-3年

  → 场景C（快速降级，概率30%）：
    双方都不想打→外交解决→恢复现状
    → 市场V型反弹
    → 但加速各方"脱钩保险"投资
```

### E-002：美元信心危机

| 影响链 | 金融链（核心）→ 能源链（石油定价）→ 所有链 |

**传导推演**：

```
T+0（触发点——假设储备份额突然跌破50%）：
  → 美债收益率飙升（+200-400bp）   [确定性: 85%]
  → DXY暴跌（-15~25%数月内）   [确定性: 80%]
  → 黄金暴涨至$4000+   [确定性: 80%]
  → 全球重新定价所有美元资产   [确定性: 85%]

T+1月（一阶后果）：
  → 美联储被迫大幅加息（保卫美元）或QE（制造通胀）→ 两难   [确定性: 80%]
  → 美国财政危机（利息支出暴涨→挤压军费/社保/基建）   [概率: 75%]
  → 新兴市场货币大幅波动（有的升有的崩）   [确定性: 80%]
  → 沙特加速人民币结算（趁势推进）   [概率: 65%]
  → CIPS交易量暴增（各国避险性加入）   [概率: 70%]

  → 场景A（美元维持主导但降级，概率50%）：
    储备份额稳在45-50%→仍是#1但不再绝对
    → "美元+"体系→多币种并存（美元+欧元+人民币）
    → 全球通胀上升（美元贬值=进口成本↑）

  → 场景B（多极货币体系形成，概率35%）：

    → 全球金融体系重组→新的IMF/世行？
    → 美国失去"嚣张的特权"→被迫像正常国家一样平衡预算

  → 场景C（美元崩溃，概率15%）：

    → 类似1971尼克松冲击但规模更大
```
