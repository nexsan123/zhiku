## 角色：崛起中的摇摆超级大国

**一句话**：14亿人口（全球#1），同时在QUAD和BRICS里，买俄罗斯油、和美国军演、和中国边境对峙——全球最大"不选边"玩家。

## 独特价值
- **人口**：14亿+年轻（中位28岁 vs 中国39 vs 美国38.5）
- **IT服务**：全球外包中心（TCS/Infosys/Wipro）
- **地理**：印度洋核心→扼守全球海运通道
- **摇摆价值**：QUAD+BRICS+SCO+买俄油+和以色列合作=被所有人需要
- **民主叙事**：西方推"民主替代中国"→印度获得大量投资/技术转让

## 关键风险
- **制造业远不如中国**：Make in India口号响但执行弱，基建/物流/官僚拖后腿
- **基础教育**：25%文盲率→人口数量优势无法完全转化为质量
- **中印边境**：加勒万冲突(2020)后仍对峙
- **水危机**：北部地下水枯竭+恒河污染

## "不选边"策略的理性计算
```
买俄罗斯折价油 → 省钱（能源进口成本↓20-30%）
加入QUAD → 从美国获得技术转让+军事合作
加入BRICS → 从中国/俄罗斯获得多极化红利
和以色列合作 → 获得先进军事技术
和伊朗合作 → 恰巴哈尔港（绕过巴基斯坦的通道）

每条线都在给印度好处。选边=失去其中几条线的好处。不选边=全拿。
→ 这不是"软弱"，是最理性的策略。
→ 直到某个事件（台海？中印冲突？）让不选边的成本>收益时才会被迫选。
```
