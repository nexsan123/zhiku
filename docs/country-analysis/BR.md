## 角色：全球南方资源超级大国

**一句话**：大豆全球#1+铁矿石（淡水河谷）+深海石油+亚马逊。中国是#1贸易伙伴，从美国后院变成中国的资源伙伴。

## 独特价值
- **大豆**：全球#1出口国（1.69亿吨/年，2025年）→ 中国60%大豆来自巴西
- **铁矿石**：淡水河谷（Vale）全球#2
- **石油**：深海盐下层（Pre-salt）= 全球增产最快的区域之一
- **亚马逊**：全球气候谈判的筹码（碳汇/生物多样性）
- **锂三角**（部分）：锂储量在增长
- **BRICS核心成员**：2024年金砖峰会主办国

## 对中国的关键意义
```
中国每年进口1亿吨+大豆
  → 60%来自巴西
  → 2018贸易战中国从美国转向巴西=成功验证
  → 巴西是中国粮食安全的"不可断的保险杠"
  → 巴西不会配合美国制裁中国（中国是#1贸易伙伴）
  → 这条关系是"利益联姻"不是意识形态同盟
```

## 关键风险
- 政策左右摇摆（Lula左/Bolsonaro右→每次换届政策大转向）
- 亚马逊砍伐→国际压力+气候风险
- 基建落后（"巴西成本"→大豆从产区到港口的物流是全球最贵之一）
- 雷亚尔波动大（新兴市场货币风险）
