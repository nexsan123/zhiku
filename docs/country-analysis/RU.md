# 俄罗斯 (RU) 战略画像

> 中等深度版 · 服务于智库量化系统
> 数据基准：2024-2025 公开数据
> 核心特征：能源超级大国 + 核武大国 + 被制裁后正滑向中国的资源附庸

## 角色：被制裁的能源大国 — 从独立极到中国的资源供应方

**一句话**：石油#3+天然气#2+核弹头#1+小麦主要出口国，但2022年后被西方制裁体系围堵，经济从多元化退化为对中国的资源依附。

## 三、战略维度分析

### 3.1 能源：唯一真实的全球杠杆

| 指标 | 数据 | 来源 | 可信度 |
|------|------|------|--------|
| 石油产量 | ~1050万桶/日 | EIA / OPEC | 0.85 |
| 天然气已探明储量 | **37.4万亿m³（全球#1）** | BP Statistical Review | 0.9 |
| OPEC+角色 | 与沙特共同领导 | — | — |
| 对欧气体出口 | 45%（2021）→ **13%（2025）** | EU Council | 0.85 |
| 对华石油出口 | **中国进口~20%**（含转口） | Kpler / Columbia CGEP | 0.8 |
| 力量1号管道 | 380→440亿m³/年（已签扩容） | OilPrice | 0.85 |
| 力量2号管道 | 规划500亿m³/年经蒙古 | CSIS / Carnegie | 0.8 |
| 力量2号状态 | 2025.9签"法律约束备忘录"，**但中方未确认，定价/融资均未定** | Carnegie | 0.8 |
| 乌拉尔原油折价 | 较布伦特折价$10-20/桶（制裁后常态） | 多方 | 0.85 |
| 油价财政盈亏点 | **~$60-70/桶** | IMF / 多方 | 0.8 |

**第一性原理**：

```
2021年的俄罗斯：
  → 石油卖给欧洲+中国+印度（分散客户）
  → 天然气管道锁定欧洲（40%+依赖）= 对欧杠杆
  → OPEC+与沙特共同定价 = 全球影响力

2025年的俄罗斯：
  → 欧洲客户从45%→13%（几乎失去）
  → 中国+印度成为"被迫的主要买家"——但他们知道俄罗斯没有替代
  → 中国故意拖延力量2号→压价→俄方议价权每天弱一分
  → 卖给中国/印度的油被迫打折$10-20
  → 从"能源武器"变成"被买方拿捏的资源出口国"

核心变化：不是产能变了，是客户结构变了
  → 客户分散=卖方有杠杆
  → 客户集中于一两个大买家=买方有杠杆
  → 俄罗斯正从前者滑向后者
```

### 3.2 军事：核武=存在性保障，常规=被乌克兰暴露

| 指标 | 数据 | 来源 |
|------|------|------|
| 核弹头 | ~5580枚（战略+战术） | SIPRI / FAS |
| 核三位一体 | 陆基ICBM+潜射SLBM+战略轰炸机 | 多方 |
| 常规军 | ~115万现役（战前）→大幅动员中 | IISS |
| 乌克兰战争损耗 | 装甲/火炮/人员大量消耗（具体数字争议大） | IISS / Oryx |
| 黑海舰队 | 多艘主力舰被乌克兰无人艇击沉/击伤 | 多方 |
| 年军费 | ~$1090亿（战时大幅增长，占GDP ~6%） | SIPRI 2024 |

**第一性原理**：

```
核武器 = 存在性保障
  → 没有国家能"消灭"俄罗斯（核武互毁保证）
  → 这使得即使常规军被削弱，俄罗斯仍是"不可忽略"的大国
  → 这是和伊拉克/利比亚的本质区别——后者没核武所以可以被推翻

常规军 = 被乌克兰暴露
  → 闪电战失败（2022年基辅方向）
  → 装甲部队损耗率远超预期
  → 后勤/通信/士气问题严重
  → 黑海舰队被无人艇这种廉价武器重创
  → 结论：俄常规军是"数量大但质量差"的苏联遗产

但注意：
  → 俄罗斯有全球最大的炮兵库存+弹药产能在恢复
  → 战时动员能力仍在（虽然代价高）
  → 西方军事援助乌克兰也在消耗自身库存
  → 消耗战中俄罗斯不一定输——取决于双方的政治意志
```

### 3.3 制裁承受力：被制裁但未崩溃

| 制裁措施 | 影响 | 俄罗斯应对 |
|---------|------|-----------|
| SWIFT部分踢除 | 跨境支付困难 | 转向CIPS+人民币结算+加密货币 |
| 油价上限$60 | 收入受限但仍在卖 | 折价卖给中国/印度/影子船队 |
| 外储冻结~$3000亿 | 失去一半外储 | 黄金增储+人民币储备 |
| 技术出口管制 | 芯片/设备/零件断供 | 从中国进口替代（洗衣机拆芯片的传闻虽夸张但方向属实） |
| 资产冻结/旅行禁令 | 寡头受影响 | 资产转移至中东/中国 |
| 对GDP影响 | 2022年GDP仅降~2.1%，2023年反弹+3.6% | 战时经济+石油收入+中国替代供应 |

**为什么没崩**：
1. 石油仍在卖（中国+印度是制裁的"漏洞"）
2. 中国提供了几乎所有被管制的技术产品替代
3. 战时经济刺激了短期增长（但不可持续）
4. 卢布被资本管制人为维稳

**但代价在积累**：
- GDP增速降至~0.9%（走向停滞）
- 人才外流50万+（IT/金融/创业者）
- 军费占GDP 6%→挤压民生
- 技术退化（汽车产业退回1990年代水平）
- 通胀持续高位

## 五、中俄关系深入分析（本体系核心关系之一）

> 详见 RELATIONSHIPS.md 第2条主线

### "无上限伙伴关系"的真实结构

```
表面叙事："平等的战略伙伴"
实际结构：中国是大股东，俄罗斯是资源子公司

证据：
  → 中国GDP $18.5T vs 俄罗斯$1.9T（10:1）
  → 中国制造业全球31.6% vs 俄罗斯<2%
  → 俄罗斯能卖给中国的：石油+天然气+木材+矿产（全是资源）
  → 中国能卖给俄罗斯的：汽车+电子+芯片+机械+消费品（全是制成品）
  → 这就是殖民时代的"资源换工业品"贸易结构

中国的策略：
  ① 故意拖延力量2号 → 俄罗斯丢了欧洲后越来越急 → 中国谈判筹码每天增加
  ② 不卖武器给俄罗斯 → 避免触发西方二级制裁 → 只卖"民用"产品
  ③ 通过BRI渗透中亚（俄罗斯传统后院）→ 俄罗斯无力阻止
  ④ 人民币在俄贸易结算占比快速上升 → 俄罗斯金融体系"人民币化"

俄罗斯的困境：
  → 被制裁后只剩中国这个大客户
  → 中国知道这一点，所以每笔交易都压价
  → 俄罗斯不敢得罪中国（唯一剩余的战略伙伴+经济命脉）
  → "无上限伙伴关系"的"无上限"只在一个方向有效
```

## 七、底层逻辑

```
俄罗斯的核心定位 = 核武大国 × 能源资源 × 领土纵深

  核武 = 存在性保障（没人能消灭俄罗斯）
  能源 = 唯一的经济杠杆（但客户结构变了→杠杆在缩水）
  领土 = 战略纵深（任何陆地入侵注定失败——拿破仑/希特勒已验证）

2022年前：三者组合 = "独立一极"（虽已衰落但仍独立操作）
2022年后：制裁切断西方市场 → 能源杠杆从"分散客户"变"中国依附"
  → 核武仍在 → 大国地位维持
  → 但经济从"资源强国"退化为"资源附庸"
  → 对中国的议价权每天在缩小

俄罗斯 vs 沙特对比：
  沙特：从美国附庸→独立摇摆国（选项在增加）
  俄罗斯：从独立一极→中国资源附庸（选项在减少）
  → 方向相反，根因：沙特客户在分散，俄罗斯客户在集中

乌克兰战争的长期影响：
  → 短期：战时经济+中国替代供应 → 没崩
  → 中期：人才外流+技术退化+军费挤压 → 增长停滞
  → 长期：取决于战争结果
    → 胜/和：恢复部分杠杆但西方制裁大概率长期维持
    → 败：政权危机风险+经济全面恶化
    → 任何结果下：对中国的依附都已结构化，难以逆转
```

### 自审

**可能过于悲观的地方**：
- 俄罗斯制裁下GDP仅降2%后反弹，比多数预测resilient得多——低估了战时经济动员能力
- 俄罗斯军工产业在战争中正在恢复——弹药/装甲车/无人机产量在增加
- 北极航道如果开通（气候变化），俄罗斯沿岸控制权是巨大的远期资产
- 俄罗斯数学/物理/航天传统不应被低估——SpaceX之前全球火箭发射靠俄罗斯
- 如果乌克兰停火+部分制裁解除，俄罗斯可能恢复对欧能源出口→客户重新分散→杠杆恢复

**但不改变底层判断**：制裁解除概率低（短中期）+人才外流不可逆+对中国的经济依附已结构化。俄罗斯从"棋手"退化为"高级棋子"——有核武这张绝对王牌，但在经济博弈中越来越像资源供应商。

---

## 综合评级：**C**

核武#1+能源储量+领土纵深是不可否认的硬资产。但制裁围堵+技术退化+人才外流+对中国依附+财政石油依赖($60盈亏线)+乌克兰消耗，使俄罗斯从"独立一极"退化为"被制裁的能源附庸"。
