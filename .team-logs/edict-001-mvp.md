# 【圣旨】智库 · 全球金融情报中枢 MVP

> 钦定日期：2026-02-17
> 颁发：司礼监掌印太监 v3.5
> 状态：已用印 ✅

## 旨意
建立 Tauri v2 桌面应用「智库」，以暗色交互式世界地图为核心界面，AI 双引擎驱动全球金融信息采集、分析与推演，通过本地 WebSocket 对接 QuantTerminal 量化项目。MVP 范围：美国金融数据全链路。

## 背景
- 皇上现有 QuantTerminal（`F:\QuantTerminal`），Tauri v2 + React + Rust + SQLite，54 个 IPC 命令，已接 Alpaca/Binance/Claude/OpenAI/Ollama
- 智库为其提供上游金融情报，两应用完全独立，本地 WebSocket 通信
- 皇上在美国拉斯维加斯，太平洋时间（PT, UTC-8），应用时间戳 PT 默认
- 硬件：RTX 4070 Super 12GB / 64GB RAM / 2TB 磁盘

## 底线（铁律，不可违逆）

### 禁止
- ❌ 禁止 AI 分析结果无法追溯到原始新闻源 URL
- ❌ 禁止未经皇上授权将 QuantTerminal 敏感数据（策略/持仓）发送给外部 AI API
- ❌ 禁止向 QuantTerminal 推送未经验证的垃圾数据
- ❌ 禁止金融板块未完成就开第二个板块
- ❌ 禁止未经实际执行就报告"已完成"
- ❌ 禁止硬编码 API Key 到代码中

### 必须
- ✅ 每条 AI 分析必须关联原始新闻链接
- ✅ API Key 加密存本地（AES-GCM + PBKDF2，复用 QuantTerminal 方案）
- ✅ 通知三级分类：P0 弹窗（突发重大）/ P1 角标（重要非紧急）/ P2 静默入库
- ✅ 双 AI 引擎可切换：Ollama 做批量摘要，Claude API 做深度分析
- ✅ 敏感数据发送给外部 API 前必须弹窗请求皇上授权
- ✅ 所有时间戳支持时区感知，默认显示 PT
