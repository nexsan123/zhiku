# 【圣旨】edict-004 · 世界局势推理引擎

> 颁发日期：2026-03-10
> 前任圣旨：edict-002-finance-reshaping（仍有效）、edict-003-ai-engine-phase-a（并行）
> 级别：XL（新架构，跨前后端，多阶段）

## 旨意

建设智库核心推理引擎——五层世界模型 + 信用周期罗盘 + 两遍情报分析法。
以全球信用/债务数据为锚，以物理现实为约束，以情报分析师级深层解读为产出，
让皇上看到的不是新闻，而是世界局势的真相。

## 背景

- 当前 `indicator_engine.rs` 仅用表面指标（VIX/F&G/联储利率）做周期判断——看症状，不看病因
- `cycle_reasoner.rs` 依赖 Claude 单次推理，无分步链，不可审计
- 新闻分析只有单条→单标签，无聚类、无深层解读、无关联分析
- BIS client 只取了 CBPOL（央行利率），信用/债务/偿债数据未采集
- 55+ RSS 新闻源已有，但产出的是新闻摘要，不是情报
- 五层世界模型和信用周期罗盘是全新架构

## 核心架构：五层世界模型

```
物理层（硬约束，不可印刷）
  能源: 油/气/铜 + EIA 库存
  粮食: FAO 食品指数 + 主粮期货
  人口: 15 国劳动人口比/中位年龄/抚养比
  → 回答: 世界的物理承载力和约束条件

信用层（金融核心引擎）
  信用周期: 15 国 BIS 数据（信用/GDP 缺口、偿债率、信用脉冲）
  美元浪潮: DXY + 联储政策 + M2 + 美债利差
  央行政策: BIS CBPOL 15 国利率方向
  → 回答: 全球信用周期在什么位置？钱在往哪流？

博弈层（大国争夺 + 美国博弈图谱）
  全球情报: RSS 新闻 → 聚类 → 深层解读
  美国博弈图谱: 6 条政策向量 × 4 组双边关系 × 决策日历 × 情景推演
  制裁/贸易壁垒: 影响资本和商品流动
  冲突热点: 影响能源/粮食供应链
  → 回答: 谁在跟谁博弈？美国下一步会打到哪个仓位？

情绪层（市场反应）
  市场: VIX/SP500/DXY
  恐贪: Fear & Greed Index
  舆情: 新闻情感分析
  → 回答: 市场定价了多少真实风险？偏差在哪？

元层（数据可信度）
  可信度: 每国每指标 reliability 评分
  交叉验证: 官方数据 vs 替代指标
  异常检测: 数据突变是否合理
  → 回答: 以上信息哪些能信？哪些存疑？
```

### 层间因果关系

```
物理层 → 信用层:
  能源紧张 → 通胀 → 央行被迫收水 → 信用周期截短
  人口老化 → 信用扩张天花板降低
  粮价暴涨 → 穷国信用危机

信用层 → 博弈层:
  美元退潮 → 新兴市场承压 → 地缘紧张加剧
  信用过热 → 资产泡沫 → 争夺优质资产的博弈加剧

博弈层(美国博弈图谱) → 信用层:
  芯片管制加码 → NVDA/AMD 营收受损 → 科技板块估值重定价
  美元武器化 → 去美元化叙事 → BTC/黄金需求上升
  加密监管收紧 → 直接冲击 crypto 持仓

博弈层 → 物理层:
  战争/制裁 → 能源/粮食供应中断
  贸易壁垒 → 供应链重构

情绪层 vs 其他层:
  偏差 = 机会或风险（物理层说紧张但市场乐观 → 风险未定价）
```

### 国家范围（15 国/地区，三圈）

### BIS 数据集

| 数据集 | API 代码 | 内容 | 频率 |
|--------|----------|------|------|
| 信用总量 | WS_CREDIT | 非金融部门总信用 / GDP | 季度 |
| 信用缺口 | WS_CREDIT_GAP | 信用/GDP 缺口（Basel III 早期预警） | 季度 |
| 偿债率 | WS_DSR | 偿债支出占收入比 | 季度 |
| 央行利率 | WS_CBPOL | 政策利率 | 月度（已有） |
| 房产价格 | WS_SPP | 房价指数 | 季度 |

### 核心计算指标（Rust，每国）

| 指标 | 计算方法 | 用途 |
|------|----------|------|
| 信用/GDP 缺口 | BIS WS_CREDIT_GAP 直接取值 | Basel III 预警（>+10% 为危险） |
| 信用增速 | 信用总量 YoY 变化率 | Juglar 周期定位 |
| 信用脉冲 | 新增信用变化 / GDP（二阶导数） | 最敏感的转折信号 |
| 偿债率 | BIS WS_DSR 直接取值 | 债务可持续性 |
| 房价/收入比 | BIS WS_SPP 趋势 | Kuznets 周期参考 |

### 六阶段定位（一看就懂）

| 阶段 | 判定条件 | 含义 |
|------|----------|------|
| 放水期 | 央行降息 + 信用脉冲转正 + 信用缺口低位 | 水龙头打开了，钱便宜了 |
| 加杠杆 | 信用加速增长 + 脉冲为正且上升 | 大家都在借钱搞事情 |
| 过热期 | 信用缺口 > +10% 或 偿债率历史高位 | 借太多了，但还没爆 |
| 收水期 | 央行加息 + 信用脉冲转负 + 新增信用减速 | 水龙头开始关了 |
| 去杠杆 | 信用负增长 + 偿债率上升 + 违约事件增加 | 还不起了，开始暴雷 |
| 出清期 | 信用触底 + 脉冲见底回升迹象 + 资产贱卖 | 最痛的时候，但快到头了 |

### 美元浪潮（传导层）

| 指标 | 来源 | 判定 |
|------|------|------|
| DXY 趋势 | Yahoo Finance（已有） | 3 月/6 月趋势方向 |
| 联储政策 | BIS CBPOL / FRED FEDFUNDS（已有） | 加息/暂停/降息 |
| M2 增速 | FRED M2SL（已有） | 流动性扩张/收缩 |
| 美债利差 | FRED DGS10 - DGS2（需新增） | 收益率曲线形态 |

判定：
- DXY 上升 + 加息 + M2 收缩 → 美元退潮（新兴市场承压）
- DXY 下降 + 降息 + M2 扩张 → 美元涨潮（新兴市场受益）

影响修正：退潮期新兴市场风险 +1 级，涨潮期 -1 级。

| 加密线 | SEC/CFTC 监管/稳定币法案/CBDC | BTC/ETH/DeFi/稳定币 直接冲击 |
| 军事线 | 军费/军售/冲突介入 | 军工股 + 地缘风险溢价 |

### 模块 4: 情景推演

每个活跃政策向量 → 2-3 个情景 + 概率 + 资产影响。

示例 — 美国对华芯片管制:

```
情景 A: 维持现状 (概率 50%)      [合理推断]
  NVDA 中性 | SMCI 中性 | BTC 无关

情景 B: 加码管制 (概率 35%)      [合理推断]
  NVDA -5~8% | SMCI -10% | BTC 微涨(避险叙事)
  传导: 管制扩大 → AI 算力供给受限 → 中国反制稀土 → 供应链震荡

情景 C: 缓和放松 (概率 15%)      [推测性]
  NVDA +3~5% | SMCI +5% | BTC 无关
  前提: 中美关系出现实质性缓和信号
```

情景概率由 Claude 每周更新（基于最新新闻 + 政策动向），必须标注置信度等级。

## 两遍情报分析法

### 第一遍：表层分类（Groq，逐条，实时）

每条新闻经 Groq 提取：

```json
{
  "surface_tag": "energy",
  "summary": "沙特减产 200 万桶...",
  "sentiment": -0.3,
  "keywords": ["沙特", "减产", "OPEC"],
  "region": ["middle_east", "global"],
  "entities": ["沙特", "OPEC", "美国"]
}
```

新增字段（相比现有 NewsSummary）：region、entities。

### 聚类（Rust，无需 AI）

按 region + entities + 48h 时间窗口聚类。
相似度阈值：共享 >= 2 个 entity 或 >= 1 个 region + 1 个 entity。
每个簇 3-20 条新闻。

### 第二遍：深层解读（Claude，按簇，每日 1-2 次）

每个簇送入 Claude，输出结构化情报：

## 置信度分级体系

### 三级标记（所有 AI 产出统一使用）

| 等级 | 范围 | 标记 | 含义 | 用户理解 |
|------|------|------|------|----------|
| HIGH | 0.80 - 1.00 | [高确信] | 多源数据交叉验证，逻辑链完整 | 可以据此做判断 |
| REASONABLE | 0.50 - 0.79 | [合理推断] | 有数据支撑但证据链不完整，存在合理替代解释 | 值得关注，但需结合其他信息 |
| SPECULATIVE | 0.00 - 0.49 | [推测性] | 基于有限信息的推演，可能存在多种解读 | 仅供参考，不可作为决策依据 |

### 置信度计算规则

信用周期定位置信度：
- 基础分 = BIS 数据完整度（有几个指标有值 / 总指标数）
- 数据可信度修正 = 该国 reliability 评分加权
- 时效性修正 = 数据滞后越久，置信度打折（6 月滞后 ×0.9，12 月 ×0.7）

深层解读置信度：
- 新闻簇大小：>= 5 条 +0.1，< 3 条 -0.1
- 信源多样性：来自 >= 3 个独立源 +0.1
- 内部一致性：新闻间是否矛盾 -0.1 per 矛盾
- AI 自评：Claude 自身评估的置信度（参考值，不盲信）

### 置信度在前端的展示

- HIGH: 绿色标签 [高确信 0.85]
- REASONABLE: 黄色标签 [合理推断 0.63]
- SPECULATIVE: 灰色标签 [推测性 0.35]

所有面板、所有 AI 产出，无一例外必须带置信度标记。

## 底线（不可违逆）

### 禁止
- 禁止硬编码任何 API Key
- 禁止将推测性解读（confidence < 0.5）标为高确信
- 禁止跳过数据可信度检查直接聚合低质量数据
- 禁止单条新闻做深层解读（必须聚类后批量分析）
- 禁止 AI 产出不带置信度标记
- 禁止伪造 BIS 数据或用 mock 数据冒充真实数据
- 禁止未经运行验证就声称"已完成"
- 禁止情景推演不标注置信度等级
- 禁止将 AI 估算的情景概率当作事实展示（必须标「AI 推演」）

### 必须
- 所有 AI 产出必须带置信度 + 三级标记（HIGH/REASONABLE/SPECULATIVE）
- 深层解读必须关联原始新闻 source_urls（ZK-01）
- 信用周期定位必须可追溯到具体 BIS 数据点
- 信用脉冲（二阶导数）必须作为核心指标之一
- 低可信度国家数据必须在前端标注警告
- 数据可信度表必须存在且至少覆盖 15 国
- 博弈图谱情景推演必须标注「AI 推演」+ 置信度 + 更新日期
- 决策日历事件必须标注影响资产和方向

## 确定性目标

- 最低目标：BIS 信用数据入库 + 15 国六阶段定位 + 美元浪潮 + 置信度标记
- 最高目标：五层完整 + 两遍分析法 + 深层解读 + 美国博弈图谱 + 情报面板 + QuantTerminal 推送
